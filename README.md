# Retail Sales Analysis Dashboard

### Project Overview
This project involves a comprehensive analysis of 1,000 retail transactions to uncover sales trends and customer behavior.

### Key Features
* **Data Cleaning:** Prepared raw data for accurate analysis.
* **Pivot Tables:** Summarized revenue by Gender, Age, and Product Category.
* **Interactive Visualization:** Created a professional dashboard with dynamic charts.
* **Slicers:** Added real-time filtering for better data exploration.

### Business Insights
* **Total Sales:** $456,000
* **Best Category:** Electronics ($156,905)
* **Top Segment:** Female customers (51% of total sales)
